Daily Planner

All the available planners are designed for widespread use, Students who are attending higher studies like college and university students have busy schedules. due to hectic schedule, students are forget about deadlines for some assignment or task they have planned to do.

In Application user can create a schedule for the daily or weekly basis and adding all the upcoming deadlines and deciding if you have completed the required work on not can be beneficial for all the students attending their higher studies.

Roles

Developer, Designer (Group Project in Postgraduate certificate course)

Application Information

React Navigation Android Application with firebase database integration

Documentation

App Functionality
  • User can add Tasks, Modify Tasks, Delete Tasks and prioritize task
  • User can add name, reminder time, priority, notes in the task
  • User can view all the tasks added for the month in calendar view
  • User can view all the task added for the day in List view
  • User can modify when the notification comes for the task (hour before, day before etc.)
  • User can add, modify and delete tasks simply and easily (within 30 sec)
  • User can view all the ads posted by them
