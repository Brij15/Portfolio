Barter Books

This is app that allows to user to post ads on their books and view ads accordingly to thier location (Barrie). Users can post books for barter with someone else or sell it with discount price. This will allow students to find textbooks, referencebooks.

Roles

Developer, Designer (Group Project in Postgraduate certificate course)

Application Information

Android Application with firebase database integration

Documentation

App Functionality
  • Splash screen with animation that automatically redirect to Login screen
  • User can login with username and password or Social media, User can register
  • Show all newest ads with filters (Category and Location) to search posts
  • View details about the ads(Images, Price, Seller details and Description)
  • User can add, edit and delete ads
  • User can view all posted posts.
